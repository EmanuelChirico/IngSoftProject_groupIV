void it.diem.unisa.ingsoft.gruppoiv.rubrica.GestioneFile.esportaContatti ( )

Il metodo esportaContatti() esporta una rubrica su un file.

Postcondizione
La lista di contatti viene salvata nel file scelto: se già esisteva un file con quel nome, viene sovrascritto.
Invariante
La lista di contatti non viene modificata.
Parametri
[in]nomeFileIl file su cui bisogna esportare la rubrica.

◆ importaContatti()

void it.diem.unisa.ingsoft.gruppoiv.rubrica.GestioneFile.importaContatti ( )

Il metodo importaContatti() importa una rubrica da un file.

Precondizione
Il file, il cui nome è passato come parametro, esiste.
Postcondizione
La rubrica che era presente nel file viene salvata in una lista di contatti, se la struttura del file è coerente con quella della rubrica.
Invariante
Il file non viene modificato.
Parametri
[in]nomeFileIl file da cui bisogna importare la rubrica
